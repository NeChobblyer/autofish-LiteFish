# autofish-LiteFish
Плагин авторыбалка для плагина LiteFish.   Как работает: 1. Скачай файл .бат  2. Начни рыбалку в майнкрафте 3. Запусти скаченный файл  Автоматическая рыбалка должна начаться после первого заброса удочки.  Я вайбкодер и это штучка написана ИИ, для майнкрафта 1.21.1, но скорее всего работает и на других версиях.
